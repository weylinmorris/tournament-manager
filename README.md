# README

### Setup

#### Install Gems
```
$ bundle install
```

#### Install React Dependencies
```
$ npm install
```

#### DB Migrations
```
$ rake db:reset db:migrate
```

#### Webpack
```
$ yarn build --watch
```

#### Tests
```
$ rspec
```

#### Development Server
```
$ rails server -b 0.0.0.0 -p <port> -e development
```
